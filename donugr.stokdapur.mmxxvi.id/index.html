Deskripsi Showcase

Clean minimal operasional untuk stok dapur

Aplikasi ini ditujukan untuk pencatatan stok bahan di dapur. Data operasional utama disimpan lokal di perangkat, aplikasi tidak mewajibkan akun, dan akses perangkat hanya dipakai saat fitur terkait digunakan.

ToS / Privacy

Kebijakan Privasi Stok Dapur

Dokumen ini menjelaskan bagaimana aplikasi mengakses, menggunakan, menyimpan, dan membagikan data pengguna serta data perangkat.

Data yang diproses

  • Data bahan, kategori, stok masuk, stok keluar, sisa stok, dan laporan inventori
  • Foto bahan yang diambil melalui kamera perangkat
  • Preferensi aplikasi seperti tema, bahasa, dan pengaturan printer

Izin dan fitur

  • Kamera untuk foto dokumentasi bahan
  • Bluetooth untuk printer
  • File dan penyimpanan untuk backup, restore, export, dan share

Penyimpanan

Sebagian besar data disimpan lokal di perangkat dan tetap berada di perangkat sampai dihapus, dipulihkan dari backup, atau aplikasi dihapus.

Pembagian data

Data operasional utama tidak dibagikan untuk iklan atau pemasaran. Data hanya diteruskan untuk share/export, atau jika diwajibkan hukum.
