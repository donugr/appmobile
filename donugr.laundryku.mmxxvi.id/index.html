Deskripsi Showcase

Local-first untuk LaundryKu

LaundryKu merupakan aplikasi manajemen usaha laundry yang dirancang untuk membantu pemilik usaha mengelola seluruh operasional sehari-hari secara lebih terstruktur, efisien, dan profesional. Aplikasi ini mendukung berbagai jenis usaha laundry seperti laundry kiloan, laundry satuan, laundry sepatu, tas, karpet, gorden, hingga laundry dengan banyak cabang. Mulai dari penerimaan cucian, pengelolaan pelanggan, pencatatan pembayaran, pelacakan status cucian, pengelolaan stok bahan, pencatatan pengeluaran, hingga penyusunan laporan usaha dapat dilakukan dalam satu aplikasi yang mudah digunakan.

ToS / Privacy

Kebijakan Privasi LaundryKu

Dokumen ini menjelaskan bagaimana aplikasi mengakses, menggunakan, menyimpan, dan membagikan data pengguna serta data perangkat.

Data yang diproses

  • Data pelanggan, transaksi, pembayaran, dan riwayat cucian
  • Preferensi aplikasi seperti tema, bahasa, dan pengaturan printer
  • Data notifikasi dan metadata teknis terbatas bila fitur notifikasi dipakai

Izin dan fitur

  • Notifikasi untuk pesan lokal
  • Push notification
  • Bluetooth untuk printer thermal
  • File dan penyimpanan untuk backup, restore, export, dan share

Penyimpanan

Sebagian besar data disimpan lokal di perangkat dan tetap berada di perangkat sampai dihapus, dipulihkan dari backup, atau aplikasi dihapus.

Pembagian data

Data operasional utama tidak dibagikan untuk iklan atau pemasaran. Data hanya diteruskan untuk notifikasi, share/export, atau jika diwajibkan hukum.
